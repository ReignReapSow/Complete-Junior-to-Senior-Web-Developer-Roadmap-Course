CREATE-REACT-APP NOTES
=============

1.CREATE react-app
    A. RUN "npm init create-react-app <appName>" OR "yarn init create-react-app <appName>"
    B. CD <appName>
    C. RUN yarm start or npm start


2. UPDATE PACKAGE in Project

    A. Go to package.json and change the packages version number to the current one
    B. RUN npm install (INSTALLS the updates)
